cu-caterpillar

Docs.rs/cu-caterpillar

This is an example for the Copper project to measure a base latency from the runtime. It can be used dry running on a host or on a Raspberry Pi flipping GPIO pins.

cu-config-gen

Repository
Docs.rs/cu-config-gen

This is an example for the Copper project to show how to programmatically generate a config.

cu-consolemon

Repository

cu-lewansoul

cu-monitoring

Repository
Docs.rs/cu-monitoring

cu-multisources

Repository
Docs.rs/cu-multisources

cu-pid

Repository
Docs.rs/cu-pid

A PID controller for the Copper project.

cu-rp-balancebot

Repository
Docs.rs/cu-rp-balancebot

This is a full robot example for the Copper project. It runs on the Raspberry Pi with the balance bot hat to balance a rod.

cu29

Repository
Docs.rs/cu29

Copper Runtime Core crate. Copper is an engine for robotics.

cu29-clock

Repository
Docs.rs/cu29-clock

Copper Robot Clock implementation. It is a monotonic high precision clock for real time applications. It has a mock feature for testing time dependent behaviors. It is part of the Copper project but can be used independently.

cu29-derive

Repository
Docs.rs/cu29-derive

This is the copper project runtime generator. It cannot be used independently from the copper project.

cu29-export

cu29-helpers

Repository
Docs.rs/cu29-helpers

This is an helper to build quickly a skeleton of an application with Copper. It cannot be used independently from the copper project.

cu29-intern-strs

Docs.rs/cu29-intern-strs

cu29-log-derive

cu29-log-runtime

cu29-soa-derive

cu29-traits

Repository
Docs.rs/cu29-traits
